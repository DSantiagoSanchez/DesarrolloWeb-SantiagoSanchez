Asylum

"Asylum" fue creado por Santiago Sanchez durante el transcurso del curso "Desarrollo de Videjuegos" de Coderhouse. Este proyecto se trata de un videojuego 3D del genero horror en primera persona, en el cual el objetivo principal es encontrar un codigo de seguridad para poder escapar. Para realizar este proyecto se utilizaron las herramientas Unity, C# y VSC (Visual Studio Code), entre otras.
